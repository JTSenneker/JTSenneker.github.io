Gordon Yasmar: Saviour of the Sunken Civilization

Primary Role: Programmer
Development Time: January 2018 - May 2018
Engine: Unity 3D
Platform: PC
Download: link

Gordon Yasmar: Saviour of the Sunken Civilization was my capstone project at Ferris State University for their Digital Animation and Game Design program. I worked with multiple students and created a vertical slice of a JRPG. I did my best to manage the project while also programming the various features of the game.

Over the course of the project I learned a multitude of things including:

  • Unity's Shuriken Particle System
  • Animating With Biped in 3DS Max
  • Unity's Mechanim Animation System
  • Unity's Timeline Animation System
  • Implementing an Inventory System Similar to the Classic Final Fantasy Series
  • Implementing a Real Time Action Based Battle System
  • Unity's AI System
  • Using Plugins With Unity
  • Parsing JSON Files for Unity
  • Creating Scripts for Unity's UI System
  • Unity's Animation Behaviour Scripts
  • Scene Management in Unity
  • Programming Scene Transitions
  • Programming Shops
